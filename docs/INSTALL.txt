twtour.i18n Installation
------------------------

To install twtour.i18n using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``twtour.i18n`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        twtour.i18n
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
